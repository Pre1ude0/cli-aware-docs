---
title: 'Getting started'
description: 'Get started with cli-aware'
---

Cli-aware, at its core, is a function that examines the `User-Agent` header of a request and returns the result of the corresponding function passed to it.

```ts
return cliAware({
    request,
    browser() {
        // Browser response
        return new Response("`curl` this page in your terminal!");
    },
    cli() {
        // cli response
        return new Response("Hello from the cli!");
    },
}) as Response;
```
